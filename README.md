## 배운 것 
- Next.js의 다양한 데이터 렌더링 방식에 대한 이해 및 활용
- 클라우드 서비스 활용을 통한 데이터 관리 및 소통
- 3rd party API를 통한 데이터 관리 및 소통
- 안정적인 코드 구축
- 일자 및 주소 검색 관련 라이브러리 활용
- Git 브랜치 전략, 코드 리뷰
- Slack, Notion 등을 통한 프로젝트 매니지먼트
- Google Analytics를 통한 제품 분석
- 복잡한 UI 요소 구현을 위해 외부 패키지 리서치 및 활용
- 각 파트를 담당하는 사람들과의 커뮤니케이션 역량 향상
- 여러 사람에게 효과적으로 업무 분배를 하고 관리
